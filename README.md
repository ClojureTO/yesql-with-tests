# yesql-examples

An example of using the [Yesql](https://github.com/krisajenkins/yesql/tree/devel) library with unit tests.

## Usage

To run the tests either run in the terminal:

    lein test

or in the REPL:

```clojure
(use 'yesql-examples.core-test)
(run-tests)
```
